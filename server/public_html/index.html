Wellcome to the simplest http server

This server works with websockets and contains the minimun required dependencies to make it work, in a proper way.

You should never use this server on production, It's designed with learning porpourses and will not be maintained in the future.

Dependencies:

  • config: contains server configuration files. Yoy may adjust server parameters there, like listening port, maximun concurrent allowed connections, ...
  • default_responses: contains the default http response pages (500, 404, ...) YOU SHOULDN'T TOUCH THIS FOLDER, if you want to add custom response pages you have to create them inside public_html > custom_responses
  • public_html: the folder in which you have to actually place your html.
